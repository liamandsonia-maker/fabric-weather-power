# fabric-weather-power
repo for weather and power data in fabric

## Overview
This project demonstrates an end-to-end analytics pipeline built in Microsoft Fabric.  
Weather data and electricity demand data are ingested, transfomred using Python (PySpark), and stored in a Fabric Lakehouse for analytics

## Architecture
- Microsoft Fabric Pipelines
- Fabric Lakehouse (OneLake)
- Python/PySpark notebooks
- SQL analytics layer

## Key features
- Multi source ingestion (weather + Power)
- Python based transformations
- Bronze -> Silver -> Gold data model
- Reproducible deployment using Git
